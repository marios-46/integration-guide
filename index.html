How to Use Torrentio Add-on Inside Stremio

A complete, beginner-friendly guide explaining how to install, configure, and use the Torrentio add-on inside Stremio for everyday streaming.

Introduction

Stremio is designed around an add-on ecosystem that allows users to extend the platform without modifying the core application. Add-ons supply catalogs, metadata, and streaming links that Stremio can display in a unified interface.

This guide explains how to use Torrentio inside Stremio in a clear and practical way. The focus is on understanding how the add-on fits into Stremio’s workflow rather than promoting any specific use case.

What Is Torrentio in the Context of Stremio?

Torrentio is a Stremio-compatible add-on that supplies streaming link references when a user opens a movie or TV series page. It does not replace Stremio or control playback.

Once installed, Torrentio becomes one of several data providers that Stremio can query when looking for available streams.

What You Need Before Using Torrentio

  • A working installation of Stremio
  • Internet connectivity
  • Basic familiarity with Stremio’s interface

No advanced configuration is required to get started, making Torrentio suitable for users who prefer minimal setup.

Installing Torrentio Inside Stremio

Installing an add-on in Stremio is straightforward. The process involves enabling the add-on so that Stremio can query it when needed.

  • Open Stremio on your device
  • Navigate to the Add-ons section
  • Locate the option to install add-ons via URL
  • Confirm installation

Once installed, Torrentio becomes active immediately and requires no restart.

How Torrentio Appears Inside Stremio

After installation, Torrentio does not appear as a separate app or menu. Instead, it works in the background.

Torrentio only becomes visible when you open a movie or series and view available streams.

Stremio lists available streams from all enabled add-ons together, clearly labeled for easy selection.

Using Torrentio When Watching Movies or Series

Using Torrentio is mostly passive. When you select a title in Stremio, the platform automatically queries installed add-ons.

If Torrentio finds matching sources, they appear in the stream list alongside other providers.
